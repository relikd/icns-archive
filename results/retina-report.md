# Retina compatibility тАУ Analysis results

How to read these results:
- The first column shows which keys were used for the icon (format: `@1x+@2x`)
- There are two test suits:
  - `retina` (@1x: red-color, @2x: test-pattern) used in: `icns @2x` and `icns @1x`
  - `retina-other` (@1x: test-pattern, @2x: yellow-color) used in: `app @1x`
  - This allows to see when an icon uses `@1x` instead of `@2x` and vice versa.


## app @2x

There is no app-@2x test because the VM has no support for Retina displays.
And there is no way to simulate Retina because `qlmanage` can't be run on `.app` bundles.
